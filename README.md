## 始め方

プロジェクトルートで実行：

```
git clone git@github.com:okazuki58/cursor-configs.git .cursor
```

## 使い方

Cursorのチャットで `/{filename}` を実行

引数をつけることができるコマンドもあるので、詳しくはファイルの中身を確認してください

## 更新方法

プロジェクトのルートにいる状態で：
```
cd .cursor && git pull origin main
```

## 注意
Github CLIを使用するコマンドがあるのでインストールしておいてください
```
brew install gh
```
